## 📈 Improvement Tips

### To Increase ATS Score:

```
1. Keywords:
   ✓ Match job description keywords
   ✓ Use industry-specific terms
   ✓ Include relevant certifications
   
2. Format:
   ✓ Use clear section headings
   ✓ Bullet points instead of paragraphs
   ✓ Consistent formatting
   ✓ Avoid tables and images
   
3. Impact:
   ✓ Start bullets with power verbs
   ✓ Quantify achievements (%, $, #)
   ✓ Focus on results, not duties
   
4. Structure:
   ✓ Contact info at top
   ✓ Professional summary
   ✓ Experience (newest first)
   ✓ Skills section
   ✓ Education
   ✓ Certifications
```

### Common ATS Killers (Avoid):

```
❌ Fancy fonts or formatting
❌ Tables or complex layouts
❌ Images or graphics
❌ Headers/footers with info
❌ Special characters
❌ Unusual colors
❌ Objectives instead of summaries
❌ Inconsistent formatting
```

## 🎓 How Scoring Works

### ATS Score Calculation:

```
When Job Description is provided:
ATS Score = (Skills × 0.25) + (Format × 0.25) + 
            (Impact × 0.25) + (Keywords × 0.25)

Without Job Description:
ATS Score = (Skills × 0.35) + (Format × 0.35) + 
            (Impact × 0.30)
```

### Skill Score:
```
Counts matching keywords from skills database
Multiplies by category weight
Returns 40-100%
```

### Format Score:
```
Evaluates: Structure, clarity, readability
Checks for: Proper formatting, clear sections
Returns: 25-100%
```

### Impact Score:
```
Counts power verbs used
Evaluates achievement focus
Returns: 35-100%
```

### Keywords Score:
```
Extracts from job description (if provided)
Compares with resume
Shows match percentage
Returns: Variable %
```
